# Student-Course-management-using-SQL

Hospital Admissions Analysis

This project analyzes a hospital admissions dataset to uncover trends in patient demographics, hospital departments, length of stay, treatment costs, patient outcomes, and 30-day readmissions. The analysis includes data cleaning, preprocessing, exploratory data analysis (EDA), and visualization to identify key patterns that can support operational and healthcare decision-making.

Key highlights:

* Cleaned and prepared the dataset for analysis.
* Explored patient demographics (age and gender distribution).
* Analyzed admissions across hospital departments and diagnoses.
* Examined treatment costs and average length of stay.
* Evaluated patient outcomes and 30-day readmission rates.
* Created visualizations to communicate key insights effectively.
